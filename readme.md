# Canary API

> Backend API for Canary Islands' Activities directory application

## Usage

Rename 'config/cofig.production.env" to "config/config.env" and update the settings.

## Install dependencies
```
npm install
```

## Run app
```
# Run in dev mode
npm run dev

# Run in prod mode
npm start
```
